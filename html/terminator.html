NAME

terminator - (cheatsheet)


SYNOPSIS

terminator


DESCRIPTION


Group/Broadcast

  Super+g       : group all
  Super+Sh+g    : end group
  Super+t       : group in tab (2 tabs must be present)
  Super+Sh+t    : end group in tab
  Alt+a         : broadcast all
  Alt+g         : broadcast group
  Alt+o         : end broadcast

Panneau/Terminal

   Ctrl-Sh-o     : split horizontal
   Ctrl-Sh-e     : split vertical
   Ctrl-Sh-w     : close panneau/terminal
   Ctrl-Alt-x    : rename panneau

Tab

   Ctrl-Sh-t     : new tab
   Ctrl-Alt-t    : rename tab
   Dbl-Click     : rename tab
   Ctrl-Sh-q     : quit all

Navigation

   Alt-→↓        : move to panel
   Ctrl-Sh-n     : move to next panel
   Ctrl-Sh-p     : move to prev. panel
   Ctrl-Tab      : move to next panel
   Ctrl-Sh-Tab   : move to previous panel
   Ctrl-PgDown   : move to next tab
   Ctrl-PgUp     : move to previous tab

Divers

   Ctrl+sh+s    : show/hide scrollbar
   Ctrl+ +      : font size ++
   Ctrl+ -      : font size --
   Ctrl+Alt+w   : rename window
   Ctrl+Sh+i    : new terminator
